================================================================================================
ANÁLISIS COMPARATIVO DE MACHINE LEARNING PARA LA DETECCIÓN DE OPERACIONES FRAUDULENTAS BANCARIAS
================================================================================================


En esta carpeta se encuentran el script en Python que construye el modelo para la detección
de las transferencias fraudulentas y los datos requeridos para su entrenamiento y evaluación.

Este es el contenido:

    - python\transferencias.py
      Programa en Python para que construye el modelo.
      
    - python\dataset\htresepa.zip
      Archivo CSV con el dataset de entrenamiento y evaluación, comprimido en un archivo .zip.
      
    - python\img
      Carpeta con imágenes generadas desde el script de Python, que ilustran diversos
      resultados intermedios.
